YesMarket
-----
## 목적
- 쇼핑몰 개발에 기본이 되는 기능들을 구현하면서 쇼핑몰 도메인을 이해.
- 개발 단계에서 기능 테스트를 위한 테스트 코드 작성.
- CI / CD를 구현해서 개발에만 집중할 수 있는 환경을 구성.
- 클라우드와 분산 서버 환경에 대한 이해.
- Git Flow 전략으로 브랜치를 관리

## 기술 스택
- Java 8
- Springboot 2.4
- Spring Data JPA
- Gradle
- MySQL 8.0
- Docker
- Jenkins
- Redis
- NginX
- AWS EC2
- AWS RDS
